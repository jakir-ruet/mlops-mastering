### Data Pipeline Orchestration

Data pipeline orchestration means managing, scheduling, and monitoring workflows that move and transform data. Instead of manually running scripts like:

- Extract data
- Clean data
- Transform data
- Load to database

> An orchestrator does it automatically, in correct order, with retries, logs, and scheduling.

#### Apache Airflow

Apache Airflow is the most widely used open-source orchestration tool.
